Definiții

  Arborele este un graf neorientat conex fără cicluri în care unul din noduri este desemnat ca rădăcină. Nodurile pot fi aşezate pe niveluri începând cu rădăcina care este plasată pe nivelul 1.
  Nodul fără nici un predecesor este numit rădăcină, nodul care nu are nici un succesor (niciun fiu) este numit frunză.Succesorii unui nod vor fi numiţi în continuare şi descendenţi.
  Gradul unui nod este numărul de fii pe care îi are acel nod. Nodurile care au gradul zero se numesc noduri frunză sau noduri terminale.
  Gradul unui arbore este valoarea maximă a gradelor nodurilor sale, sau, cu alte cuvinte, gradul unui arbore este egal cu numărul maxim de fii pe care îi are un nod.
 Prorietate: Dacă între 2 noduri ale arborelui există un drum atunci acel drum este unic.
 Inălţimea arborelui este lungimea maximă a drumurilor din arbore.
 Nivelul unui nod este egal cu nivelul tatălui sau la care se adaugă 1. Nivelul rădăcinii este egal cu 0.
 Două noduri avînd acelaşi nivel şi acelaşi tată se mai numesc şi fraţi.
 Observaţie : Inălţimea unui arbore este egală cu nivelul maxim al frunzelor sale.
 Arborele echilibrat este acel arbore în care diferenţa dintre lungimile drumurilor de la nodul rădăcină la oricare nod frunză este cel mult 1.
 Un arbore ordonat este un arbore în care fii fiecărui vîrf sunt ordonaţi.
